常见参数

  • model string：模型名称
  • messages array：历史对话消息
    • role string：
      • user：用户输入的消息
      • system：系统提示消息
      • assistant：历史消息
  • stream boolean：是否为流式输出

响应数据

  • id string：调用唯一id
  • choices array：生成内容数组
  • create integer：生成时间戳
  • model string：调用模型名称
  • usage object：token信息

核心api

  • 文档加载器

    • lFileSystemDocumentLoader, 根据本地磁盘绝对路径加载
    • lClassPathDocumentLoader，相对于类路径加载
    • lUrlDocumentLoader，根据url路径加载

  • 文档解析器

    • lTextDocumentParser，解析纯文本格式的文件
    • lApachePdfBoxDocumentParser，解析pdf格式文件
    • lApachePoiDocumentParser，解析微软的office文件，例如DOC、PPT、XLS
    • lApacheTikaDocumentParser（默认），几乎可以解析所有格式的文件

    配置pom依赖

  • 文档分割器

    • lDocuemntByParagraphSplitter，按照段落分割文本

    • lDocumentByLineSplitter，按照行分割文本

    • lDocumentBySentenceSplitter，按照句子分割文本

    • lDocumentByWordSplitter，按照词分割文本

    • lDocumentByCharacterSplitter，按照固定数量的字符分割文本

    • lDocumentByRegexSplitter，按照正则表达式分割文本

    • lDocumentSplitters.recursive(…)(默认)，递归分割器,优先段落分割，再按照行分割，再按照句子分割，再按照词分割
